# Screenshots

No PNG screenshots were saved to this folder. The browser-verification tool used for
the P0 spikes can display screenshots inline during the session, but this agent had no
available tool to persist those images as files to disk. All visual findings from the
screenshots are instead described and quoted (via direct DOM/computed-style
inspection) in `../VERDICTS.md`, which is reproducible by opening each spike's
`index.html` in a browser yourself.
